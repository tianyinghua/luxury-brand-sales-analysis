# Luxury Brand Sales & Marketing Efficiency Analysis

This project analyzes sales revenue and marketing expenditure data for four luxury brands: Chanel, Dior, Gucci, and Louis Vuitton.

The analysis was conducted using Excel pivot tables and basic calculations to compare:

- Sales growth
- Marketing growth
- Marketing efficiency

The results highlight differences in marketing effectiveness and sales performance across luxury brands.
